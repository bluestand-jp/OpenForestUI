# オーバーレイのフォント

チャンピオンセレクトオーバーレイで使用する TrueType フォント。[`../../europe/style/fonts.less`](../../europe/style/fonts.less) で `@font-face` ファミリーとして宣言され、[`../../europe/style/`](../../europe/style/README-ja.md) の LESS を通じて適用される（例: `body { font-family: Rawline }`）。

## Contents

| File | `@font-face` family | Used for |
| --- | --- | --- |
| `Rawline-Medium.ttf` | `Rawline` | オーバーレイ本文のデフォルトフォント（プレイヤー／チーム名、タイマー）。 |
| `Raleway-Regular.ttf` | `Raleway` | 補助テキスト用フォント。 |
| `amarurgt.ttf` | `Amarello` | ディスプレイ／アクセント用フォント。 |
