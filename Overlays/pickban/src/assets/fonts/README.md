# Overlay fonts

TrueType fonts used by the champion-select overlay. They are declared as `@font-face` families in [`../../europe/style/fonts.less`](../../europe/style/fonts.less) and applied through the LESS in [`../../europe/style/`](../../europe/style/) (e.g. `body { font-family: Rawline }`).

## Contents

| File | `@font-face` family | Used for |
| --- | --- | --- |
| `Rawline-Medium.ttf` | `Rawline` | Default overlay body font (player/team names, timer). |
| `Raleway-Regular.ttf` | `Raleway` | Secondary text font. |
| `amarurgt.ttf` | `Amarello` | Display/accent font. |
