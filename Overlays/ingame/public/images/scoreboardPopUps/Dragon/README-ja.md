# ドラゴンポップアップアート

属性ドラゴンごとの spawn/kill/soul ポップアップバナー。ドラゴンタイプごとに 1 サブディレクトリ。消費側（[`../../../../src/visual/ObjectivePopUpVisual.ts`](../../../../src/visual/ObjectivePopUpVisual.ts)）はポップアップタイプから `Kill`/`Spawn`/`Soul` を取り除いてフォルダを選び、`<type><Event>.{png,mp4}` を読み込む。Elder に Soul バリアントはない。
