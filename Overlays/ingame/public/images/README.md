# Overlay icons and themed art

General-purpose overlay icons plus themed subtrees for dragons, lanes, the broadcast skin and the scoreboard popup videos. Loaded as `frontend/images/...`, mostly preloaded in [`../../src/scenes/IngameScene.ts`](../../src/scenes/IngameScene.ts).

## Contents

- `ObjectiveGold.png`, `ObjectiveCdr.png` — gold/CDR objective-bar icons.
- `ScoreboardGold.png`, `ScoreboardCenterIcon.png` — scoreboard gold and center icons.
- `tower.png`, `baronTimer.png` — tower and baron-timer glyphs.
- `InfoTabSeparator.png` — info page tab separator.

## Subdirectories

| Dir | Purpose |
| --- | --- |
| [`dragons/`](dragons/) | Per-dragon-type icons (scoreboard + timer variants) |
| [`lanes/`](lanes/) | Lane glyph SVGs (top/mid/bot) |
| [`prm/`](prm/) | Broadcast skin objective icons |
| [`scoreboardPopUps/`](scoreboardPopUps/) | Objective spawn/kill/soul popup videos and stills |
