# "Europe" ドラフトレイアウト

欧州（EU）風の "europe" レイアウトで実際のチャンピオンセレクトオーバーレイを描画する React コンポーネント群。中央カラム（ロゴ、パッチテキスト、ドラフトタイマー）を、ブルー／レッド両チームのピックとバンが挟む構成。フォークが提供する唯一のレイアウト。

## Contents

| File | Role |
| --- | --- |
| `Overlay.jsx` | トップレベルのレイアウトコンポーネント。チャンピオンセレクト開始時に段階的なリビールアニメーションを再生し、タイマー／ロゴの中央ボックスと両チーム（ピック + バン + 名前／スコア／コーチ）を描画し、CSS 変数経由でチームカラーを適用し、「未接続」バナーを表示する。 |
| `Pick.jsx` | 1 つのピックスロット: チャンピオンのローディングスプラッシュ、サモナースペルアイコン（有効かつアクティブにピック中でないとき）、プレイヤー表示名。 |
| `Ban.jsx` | 1 つのバンスロット: バンされたチャンピオンの正方形アイコン。 |
