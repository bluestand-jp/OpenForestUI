# Dragon-type icons

Per-elemental-dragon icon art, split by where it is rendered: large icons for the scoreboard and small icons for the respawn timer. One file per dragon type (fire, mountain, cloud, ocean, elder, hextech, chemtech), preloaded in [`../../../src/scenes/IngameScene.ts`](../../../src/scenes/IngameScene.ts).

## Subdirectories

| Dir | Purpose |
| --- | --- |
| [`scoreboard/`](scoreboard/) | Large per-dragon icons for the scoreboard (`*Large.png`) |
| [`timers/`](timers/) | Small per-dragon icons for the respawn timer (`*Timer.png`) |
