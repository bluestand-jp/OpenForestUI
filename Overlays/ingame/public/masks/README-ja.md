# オーバーレイのアルファ/ビットマップマスク

オーバーレイ面を意図したシルエットにクリップするマスクテクスチャ（Phaser のビットマップ/ジオメトリマスク）。[`../../src/scenes/IngameScene.ts`](../../src/scenes/IngameScene.ts) で `masks_*` としてプリロードされ、対応するビジュアル要素が適用する。

## 内容

- `ChampCoverLeft.png`、`ChampCoverRight.png` — 左/右のチャンピオンポートレートカバー用マスク。
- `ScoreboardMask.png` — スコアボード面をクリップ。
- `ItemTextLeft.png`、`ItemTextRight.png` — 左/右のアイテムテキストパネルをクリップ。
- `InfoPage.png` — インフォページ面をクリップ。
- `Graph.png` — gold/グラフ面をクリップ。
- `PopUpMask.png` — オブジェクティブポップアップバナーをクリップ（`ObjectivePopUpVisual`）。
