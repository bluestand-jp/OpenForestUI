# 共有データモデル

ソリューション全体で共有されるプレーンなデータクラス。2 つのグループに分かれる: オーバーレイやプロバイダーがシリアライズするアプリ向け DTO と、ゲームデータの解釈に使う Riot/Community Dragon の静的データ型（例: Farsight が読み取った生の XP をチャンピオンレベルへ変換、あるいはアイテムのゴールド値）。
