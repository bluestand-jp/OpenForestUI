# Overlay alpha/bitmap masks

Mask textures that clip overlay surfaces to their intended silhouette (bitmap/geometry masks in Phaser). Preloaded as `masks_*` in [`../../src/scenes/IngameScene.ts`](../../src/scenes/IngameScene.ts) and applied by the corresponding visual elements.

## Contents

- `ChampCoverLeft.png`, `ChampCoverRight.png` — masks for the left/right champion portrait covers.
- `ScoreboardMask.png` — clips the scoreboard surface.
- `ItemTextLeft.png`, `ItemTextRight.png` — clip the left/right item-text panels.
- `InfoPage.png` — clips the info page surface.
- `Graph.png` — clips the gold/graph surface.
- `PopUpMask.png` — clips the objective popup banner (`ObjectivePopUpVisual`).
