# 放送用スキンアイコン

放送用スキンのオブジェクティブアイコン。比較ボトムバー/スコアボードに描画される（[`../../../src/visual`](../../../src/visual) の `PrmBottomBarVisual`、`PrmScoreboardVisual`）。[`../../../src/scenes/IngameScene.ts`](../../../src/scenes/IngameScene.ts) で `prm_*` としてプリロードされる。

## 内容

- `dragon.png`（`prm_dragon`）、`baron.png`（`prm_baron`）、`grub.png`（`prm_grub`）、`tower.png`（`prm_tower`）、`gold.png`（`prm_gold`） — 放送用 top/bottom バー用オブジェクティブカウンターアイコン。
- `flags/` — 現在は空（チーム/国別のフラグアート用に予約）。
